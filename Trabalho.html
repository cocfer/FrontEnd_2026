Utilização

Repetir uma ação um número determinado ou indeterminado de vezes

Laços de repetição

Laço FOR

Quando o numero de repetições é informado

É utilizado quando o número de repetições é informado ou para percorrer uma lista

Para mais informações clique aqui

Laço WHILE

Primeiramente testa uma variavel antes do laço e caso nao atender o laço não é iniciado

É utilizado para repetir uma parte código enquanto uma condição específica for verdadeira

Para mais informações clique aqui

Laço DO-WHILE

Executa e depois testa a variavel e sempre é executado pelo menos uma vez

É utilizado quando uma parte do código precisa ser executada uma vez mesmo condição ser verdadeira ou falsa

Para mais informações clique aqui
